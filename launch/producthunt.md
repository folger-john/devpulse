# Product Hunt Launch

**Name:** DevPulse
**Tagline:** 25 free developer tools in one fast, private place
**Description:**

DevPulse is a collection of 25 free online developer utilities — JSON formatter, Base64 encoder, UUID generator, hash generator, JWT decoder, regex tester, QR code maker, and more.

Every tool is:
✅ Free forever — no signup, no paywall
✅ Private — no data stored, no tracking
✅ API-enabled — REST endpoints for every tool
✅ Open source — MIT licensed on GitHub

Built for developers who are tired of bloated tool sites with cookie banners and popups.
