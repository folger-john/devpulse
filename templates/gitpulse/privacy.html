Privacy Policy

Last updated: April 4, 2026

Summary: GitPulse does not collect, transmit, or share your personal data. Your GitHub access token and dashboard preferences stay on your device. We use Google AdMob for ads, which has its own data collection practices.

1. Data We Collect

Data stored locally on your device:

  • GitHub access token: Your personal access token or OAuth token is stored securely on your device to authenticate with the GitHub API.
  • Dashboard preferences: Your selected repositories, notification filters, and display settings.
  • Cached data: Repository info, PR lists, and issue data cached locally for offline viewing and faster loading.

Data we do NOT collect:

  • We do not collect or transmit your GitHub token to any server other than GitHub's API.
  • We do not access your private repositories beyond what your token permits.
  • We do not track your usage, repositories, or activity.
  • We do not have analytics or telemetry.

2. Third-Party Services

GitHub API

The App communicates directly with the GitHub API using your personal access token. Your use of the GitHub API is governed by GitHub's Terms of Service and Privacy Statement.

Google AdMob

The App displays advertisements via Google AdMob. Google may collect device identifiers, IP addresses, and usage data for ad targeting and measurement. This is governed by Google's Privacy Policy.

3. Data Security

  • Your GitHub access token is stored securely using Android Keystore encryption.
  • All communication with the GitHub API uses HTTPS encryption.
  • The app communicates directly with GitHub and does not proxy your data through any intermediary server.

4. Data Retention

All data is stored locally on your device. Uninstalling the App deletes all stored data including your GitHub token, cached repositories, and preferences.

5. Children's Privacy

The App is not directed at children under 13. We do not knowingly collect information from children.

6. Your Rights

Since all data is stored locally on your device, you have full control. You can revoke your GitHub token, clear cached data, or uninstall the App to remove all data.

7. Changes to This Policy

We may update this Privacy Policy from time to time. The updated version will be accessible within the App and at this URL.

8. Contact

For questions about this privacy policy, contact us at support@byteoven.com
