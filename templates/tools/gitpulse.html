GitPulse

GitHub Dashboard for Android

Stay on top of your GitHub activity from anywhere. Browse repos, check commits, track issues and PRs, and get notifications - all in a clean mobile interface. Your token is encrypted on-device with EncryptedSharedPreferences.

Download APK Features

Features

📊

Activity Dashboard

See your recent GitHub activity at a glance. Commits, issues, PRs, and stars in a unified feed.

📁

Repository Browser

Browse your repos with search and filter. View files, README, and repo stats.

🔀

Commits, Issues & PRs

View commit history, open issues, and pull requests for any repository. Tap to see details.

🔔

GitHub Notifications

Check your GitHub notifications without opening the browser. Mark as read directly from the app.

👤

User Profile & Stats

View your profile, follower counts, public repos, and contribution stats.

🔐

Encrypted Token Storage

GitHub token stored with EncryptedSharedPreferences (AES-256). Backed by Android Keystore.

🌙

Catppuccin Theme

Beautiful dark Mocha theme. Designed to feel at home next to your terminal.

🔒

Privacy First

No analytics, no telemetry. Token never leaves your device. All API calls go directly to GitHub.

⚡

Fast & Lightweight

Minimal permissions, small APK size. Built with Kotlin and Jetpack Compose.

Security

  • ✓ GitHub token encrypted with EncryptedSharedPreferences (AES-256-SIV + AES-256-GCM)
  • ✓ Encryption key stored in Android Keystore (hardware-backed where available)
  • ✓ All API calls over HTTPS directly to api.github.com
  • ✓ No analytics, no telemetry, no tracking
  • ✓ Android backup disabled - token can't be extracted via adb
  • ✓ ProGuard/R8 obfuscation in release builds
