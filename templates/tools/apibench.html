APIBench

Mobile REST API Client for Android

A fast, lightweight API testing tool built for mobile developers. Send requests, inspect responses, organize collections - all from your phone. No account required, no cloud sync, your data stays on your device.

Download APK Features

Features

🌐

All REST Methods

Full support for GET, POST, PUT, PATCH, and DELETE requests. Test any endpoint from your device.

📋

Headers Editor

Add, edit, and manage custom request headers. Authorization, Content-Type, and any custom header you need.

📝

Body Editor

Compose request bodies in JSON or form-data format. Switch between content types with a tap.

🎨

JSON Highlighting

Syntax-highlighted JSON responses for easy reading. Collapsible sections for large payloads.

📁

Collections

Save and organize requests into collections. Group by project, API, or however you work.

🕐

Request History

Full history of past requests. Re-send any previous request with one tap.

📊

Response Viewer

View status code, response time, body size, and full response headers at a glance.

🌙

Catppuccin Theme

Beautiful dark Mocha theme. Easy on the eyes during late-night debugging sessions.

🔒

Local Storage Only

No cloud sync, no account required. Your API keys and requests stay on your device.
