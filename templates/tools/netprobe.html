NetProbe

Free IP & Port Scanner for Android

A lightweight network diagnostic tool for sysadmins and developers. Scan ports, discover devices on your network, ping hosts, and look up domain info — all from your phone. No account required, no data collected.

Download APK Features

Features

🔍

Port Scanner

TCP connect scan with concurrent connections. Presets for common, web, database, and full range (1-1024). Service name detection for 50+ well-known ports.

📡

Network Discovery

Scan your local /24 subnet to find all devices. Shows IP, hostname, and open ports for each discovered device.

🏓

Ping

ICMP ping with parsed statistics — packets sent/received, loss %, and min/avg/max RTT latency.

🌐

Whois / RDAP Lookup

Look up domain and IP registration info via RDAP protocol. See registrar, nameservers, and status.

📋

Device Network Info

View your device's IP, gateway, DNS servers, MAC address, network type, and external IP at a glance.

📜

Scan History

All scan results saved locally with timestamps and duration. Tap to view details, swipe to delete.

⚡

Fast & Concurrent

Parallel scanning with configurable concurrency (100 ports / 50 hosts simultaneously). Scans complete in seconds.

🌙

Dark Theme

Beautiful Catppuccin Mocha dark theme. Easy on the eyes for late-night network troubleshooting.

🔒

Security First

Input validation prevents injection attacks. No data leaves your device. Cleartext only for private network ranges.
